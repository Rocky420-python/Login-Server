# Mini Login Server

This is a **mini login server** built with **Python**, **Flask**, **bcrypt**, and **OpenAI API integration**. It is designed for educational purposes and small projects. Users can register a username/password, login securely, and interact with an AI model via API requests.

---

## Features

- Secure password storage with **bcrypt** hashing.
- Login verification.
- AI-powered responses using OpenAI API.
- Environment variables support (`.env`) for safe API key storage.
- JSON-based user database for simplicity.
- Mini API endpoints for text responses.
